Python Installation Guide

All my calculators run on Python. I will not make the calculators public as an exe, so you need to be able to run the python files yourself.

You can run the programs either through a main terminal or through a code editor.

If you're going to use a main terminal, this site gives a great tutorial:
w3schools.

There are several options for code editors, I use Spyder, but Visual Studio Code is probably better.
